git本地操作

首先在桌面创建一个文件夹：

mkdir Demo

进入文件夹

cd Demo

在文件夹下创建一个Demo.txt 的文件

touch Demo.txt

初始化git仓库

git init

git add .

git commit -m “第一次提交的log”

接下来详细讲解checkout 和 commit 的区别

checkout 是从本地的代码仓库 clone一版。

commit 是将workspace 中想要保存的代码进行提交到本地代码库

对文件进行编辑 用vi编辑器

vi Demo.txt

i进入编辑模式 esc退出

wq 保存 新增了一些内容

git commit -a

这时候会进入到vi编辑区，i进入编辑模式，第一行为对更改内容进行说明 esc + 大写Q 改为命令模式 然后 wq退出

git status 查看当前变更项
git log 查看提交信息（此信息无回滚）
git reflog 查看提交信息（有本地所有操作，可恢复）
