“=”与OC区别

区间运算符

a..<b a…b

特殊运算符解决运算中值溢出的问题 &

a& + b

复合空运算符 ??

表达式 a??b ，意思是 当a有值，将a解封 没有返回b
条件1: 表达式必须是可选类型
条件2：默认值b的了性必须要和 a 存储的类型保持一致。

b、表达式

as! 强制类型转换

let calcVC = destinationViewController as CalclatorViewController
as? 不确定是不是这个类型的，又怕程序崩溃。 返回可选值 as? returns an Optional
if let calcVC = destinationViewController as? calculatorViewcontroller{…}

断言

x = 0
assert(x != 0, “X不可能为零”)

类型约束：

不是所有的Swift类型都可以用等式==进行比较，例如：如果自己创建的类或者结构体表示一个复杂的数据模型，那么Swift无法推断这个“相等”意味着什么。
Swift标准库中定义了一个Equatable 协议 协议要求符合该协议的类型必须实现==
从而能够对符合该协议的类型的任意两个值进行比较。所有的Swift标准类型自动支持Equatable
